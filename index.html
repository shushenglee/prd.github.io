总和为4

解答：实际上是一个排序问题，算法核心在于最小值要与次小值为一组。

return sum(sorted(nums)[::2])

题目766：左下角到右下角的对角线具有相同的元素的矩阵，叫做toeplitz。

比如[[1,2,3,4],[5,1,2,3],[9,5,1,2]]

输出为真

1234 0.0 0.1 0.2 0.3

5123 1.0 1.1 1.2 1.3

9512 2.0 2.1 2.2 2.3

i-j=1 i-j=0 i-j=-1 i-j=-2

思路：首先应该考虑矩阵中的元素位置，最左下角和最右上角的值对输出没有影响，主要考虑矩阵中相同数值的位置。先判断矩阵的行和列大小，i=j,i-1=j。但是这个思路需要考虑太多的if条件判断。我们考虑规律就是判断一个元素和左上角元素是否相等。m[i,j]=m[i+1,j+1]即可。

解答： for i in range(len(m)-1):

for j in range(len(m[0])-1):

if m[i,j] != m[i+1,j+1]:

return False

return True
