Unlabeled Samples Generated by GAN Improve the Person Re-identification Baseline in vitro

发表于 2018-03-20 | 分类于 Deep Learning | | 阅读次数:

Author: Zhedong Zheng, Liang Zheng, Yi Yang

Year: 2017

文章概况

贡献

1.提出了对于离群数据的标签平滑正则化的方法（LSRO），对于校准统一的无标签图像的类别分布，可以规范监督学习模型并且提高baseline，并且性能优于两种处理未标记数据的策略。

2.作者采用了深度卷积生成对抗网络（DCGAN）来生成样本，使用基本的卷积神经网络作为特征学习。

3.引入了半监督的学习流程，将GAN生成的图像整合方法体外CNN的学习机器中，实验证明在三个person re-id数据集和一个细粒度识别数据集上性能有提升。

存在的问题与挑战

1.GANs的目前研究主要是考虑了有或者没有半监督学习在体内的样本产生的质量，而将生成的样本移除并在当前可用的学习框架下使用它们还是一个问题。（主要讨论的问题，in vitro）

2.在基于CNN的方法中使用标记和未标记的数据执行半监督学习的挑战仍然存在。

3.在特定的问题（Person Re-id）中，给每一个行人边界框分配一个ID标签所带来的数据标注代价太高。

本文做了三件事

1）利用GAN生成无标签的数据

2）提出了标签平滑正则化用于未标记数据整合的异常值（LSRO）

3）在三个person re-id数据集上的性能提升

相关工作

1.GAN（生成对抗网络）

​ GAN是要学习两个子网络：产生器和鉴别器。鉴别器判断样本是产生的还是真实的，产生器用来产生样本来欺骗鉴别器。

2.半监督学习

​ 半监督学习是监督学习的一个子类，它把未标记的样本也考虑在内，特别是标注的样本特别少的情况。不同于以前的半监督学习工作，作者采用一种不同的正则化方法来为生成的样本分配统一的标签分布。

3.Person Re-identification

​ 一些开创性的工作主要把目光放在了寻找具有区分性的手工特征上。最近在行人再识别方面取得的进展主要是提升CNN。

本文方法

1.神经网络的使用

​ 文章使用了GAN和CNN的结合，除了将CNN的最后一层进行了改动，网络中的其他部分都不发生改变。CNN采用了ResNet-50模型，输入是生成的图像和原始训练集的混合，也就是说标记和未标记的数据同时被训练。首先网络在ImageNet上预训练，然后修改最后一个完全连接层，K个神经元代表K个类别。（注意：新产生的数据并没有被分配成额外的类别，而是在现有的类别上统一地进行了标签分布）

实验部分

数据集

Market-1501：是从六台摄像机收集的大型Person Re-ID数据集。 它包含19,732张测试图像和12,936张训练图像。 这些图像由变形部件模型（DPM）自动检测，因此失调很常见，并且数据集接近实际设置。 训练集中有751个ID，测试集中有750个ID。 训练集中每个ID有17.2幅图像。

CUHK-03：包含1,467个ID的14,097张图片。香港中文大学校园内的两台摄影机会拍摄每个ID。 该数据集包含两个图像集，一个由手绘边界框注释，另一个由DPM检测器产生。 训练集中每个身份有9.6幅图像。 我们报告20次训练/测试后的平均结果。 我们使用single-shot设置。

DukeMTMC-reID：是新发布的多目标多摄像头行人追踪数据集[30]的子集。原始数据集包含来自八个不同摄像机的八个85分钟高分辨率视频。我们使用一个子集用于基于图像的re-ID，格式为Market-1501数据集。我们每隔120帧从视频中裁剪出行人图像，产生总数为36,411的边界框，用于re-ID的DukeMMC-reID数据集有1812个ID在8个相机中。在两个以上的摄像机中出现了1,404个ID，而408个ID只出现在了一个摄像机中（distractor ID）。我们随机选择702个ID作为训练集，其余702个ID作为测试集。在测试集中，我们为每个相机中的每个ID选择一个查询图像，并将剩余的图像放入图库中。结果，我们获得了具有702个ID的16,522个训练图像，其他702个ID的2,228个查询图像和17,661个图像。

Baseline方法

CNN：我们采用CNN重新识别基线。 具体而言，Matconvnet 包被使用。 在训练过程中，我们使用ResNet-50模型并修改完全连接图层，分别为Market-1501，DukeMTMC reID和CUHK03提供751,702和1,367个神经元。 所有的图像被调整为256×256，随机水平翻转、随机剪切成224×224。 我们在最后的卷积层之前插入一个dropout层，并将CUHK03的丢失率分别设置为0.5；Market-1501和DukeMMC-reID分别为0.75，我们使用momentum为0.9的随机梯度下降。 卷积层的学习率（learning rate）设置为0.002，并且在40个epochs后衰减到0.0002，并且在第50个epochs之后我们停止训练。 在测试过程中，我们在224×224输入图像的最后一个卷积层中提取2048个暗含CNN的嵌入。 两幅图像之间的相似度排序通过余弦距离来计算。
